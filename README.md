# FireWatch03

## Descrição

O FireWatch03 é um projeto acadêmico desenvolvido para aplicar na prática conceitos de desenvolvimento web e segurança utilizando tecnologias como Java, Spring Boot, Spring Security e Docker. O sistema simula um sistema de gerenciamento de informações sobre focos de incêndio, implementando uma API REST para interação e autenticação de usuários.

## Tecnologias

* Java 21
* Spring Boot
* Maven
* Oracle Database 
* Docker
